# Сервис для поиска поставщиков продуктов питания

Задача
Разработать сервис, который помогает искать и сравнивать поставщиков продуктов питания. Например: ингредиентов, готовой продукции, упаковки и других товаров, связанных с food-направлением. Сервис должен помогать пользователю быстрее находить подходящих поставщиков, собирать по ним ключевую информацию и принимать решение, с кем лучше связаться.
